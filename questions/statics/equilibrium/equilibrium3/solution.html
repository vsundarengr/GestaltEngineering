First you have to compute the vector expressions for the tensions $T_B$ , $T_C$ and $T_D$. This can be done by multiplying the tension magnitude by the unit vector that follows the path of the tension. To compute the unit vector all you have to due is divide the length of the vector by itself.
    Computing the tension vectors:
  • $\vec{T}_B = (T_B)(\vec{U}_B)$. let the unit vector $\vec{U}_B = \frac{(L_3-L_1)\hat{i}+L_1\hat{j}+2L_1\hat{k} } {\sqrt{(L_3-L_1)^2+(L_1)^2+(2L_1)^2} }$

    • So the tension vector $\vec{T}_B = (T_B)\frac{(L_3-L_1)\hat{i}+L_1\hat{j}+2L_1\hat{k} }{\sqrt{(L_3-L_1)^2+(L_1)^2+(2L_1)^2} }$

  • $\vec{T}_C = (T_C)(\vec{U}_C)$. let the unit vector $\vec{U}_C = \frac{L_1\hat{i}+(L_1-L_3)\hat{j}+2L_1\hat{k} }{\sqrt{(L_1)^2+(L_1-L_3)^2+(2L_1)^2} }$

    • So the tension vector $\vec{T}_C = (T_C)\frac{L_1\hat{i}+(L_1-L_3)\hat{j}+2L_1\hat{k} }{\sqrt{(L_1)^2+(L_1-L_3)^2+(2L_1)^2} }$

  • $\vec{T}_D = (T_D)(\vec{U}_D)$. let the unit vector $\vec{U}_D = \frac{L_1\hat{i}+(L_3-L_1)\hat{j}+2L_1\hat{k} }{\sqrt{(L_1)^2+(L_3-L_1)^2+(2L_1)^2} }$

    • So the tension vector $\vec{T}_D = (T_D)\frac{L_1\hat{i}+(L_3-L_1)\hat{j}+2L_1\hat{k} }{\sqrt{(L_1)^2+(L_3-L_1)^2+(2L_1)^2} }$

  • Now that you have all of the force vectors, set up the static equilibrium force equations.
    Setting up static equilibrium force equations:
  • $\Sigma{F}_x= 0 \implies -\vec{T}_B\hat{i}+\vec{T}_D\hat{i}+\vec{T}_C\hat{i}=0$
  • $\Sigma{F}_y= 0 \implies \vec{T}_B\hat{j}+\vec{T}_C\hat{j}-\vec{T}_D\hat{j}=0$
  • $\Sigma{F}_z= 0 \implies \vec{T}_B\hat{k}+\vec{T}_D\hat{k}+\vec{T}_C\hat{k}-weight = 0$

    • Now you can solve for the tension in each cable with the assumption that one of the cables is going to holding the maximum tesnion just before it breaks.

    Lets assume that $T_B$ will holding the max tension, that is ${{params.tmax}} {{params.unitsForce}} $

    With this assumption we can set $T_B = {{params.tmax}} {{params.unitsForce}}$. This leaves us with three equations and three unkowns.

    Solving equations using our assumption we get:

  • ${T}_B = {{params.tmax}} {{params.unitsForce}}$
  • ${T}_D = {{params.tmax}} {{params.unitsForce}}$
  • ${T}_C = 0 $
