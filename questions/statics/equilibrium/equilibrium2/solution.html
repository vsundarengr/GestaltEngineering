First you have to compute the vector expressions for the tensions $T_B$ , $T_C$ and $T_D$. This can be done by multiplying the tension magnitude by the unit vector that follows the path of the tension. To compute the unit vector all you have to due is divide the length of the vector by its self.
    Computing the tension vectors:
  • $\vec{T}_B = (T_B)(\vec{U}_B)$. let the unit vector $\vec{U}_B = \frac{0\hat{i}-L_3\hat{j}+L_2\hat{k}}{\sqrt{(L_3)^2+(L_2)^2}}$

    • So the tension vector $\vec{T}_B = (T_B)\frac{0\hat{i}-L_3\hat{j}+L_2\hat{k}}{\sqrt{(L_3)^2+(L_2)^2}}$

  • $\vec{T}_C = (T_C)(\vec{U}_C)$. let the unit vector $\vec{U}_C = \frac{\frac{L_1}{2}\hat{i}+L_1\hat{j}+0\hat{k}}{\sqrt{(\frac{L_1}{2})^2+(L_1)^2}}$

    • So the tension vector $\vec{T}_C = (T_C)\frac{\frac{L_1}{2}\hat{i}+L_1\hat{j}+0\hat{k}}{\sqrt{(\frac{L_1}{2})^2+(L_1)^2}}$

  • $\vec{T}_D = (T_D)(\vec{U}_D)$. let the unit vector $\vec{U}_D = \frac{-\frac{L_1}{2}\hat{i}+L_1\hat{j}+0\hat{k}}{\sqrt{(\frac{L_1}{2})^2+(L_1)^2}}$

    • So the tension vector $\vec{T}_D = (T_D)\frac{-\frac{L_1}{2}\hat{i}+L_1\hat{j}+0\hat{k}}{\sqrt{(\frac{L_1}{2})^2+(L_1)^2}}$

  • Now that you have all of the force vectors, set up the static equilibrium force equations.
    Setting up static equilibrium force equations:
  • $\Sigma{F}_x= 0 \implies \vec{T}_D\hat{i}-\vec{T}_C\hat{i}=0$
  • $\Sigma{F}_y= 0 \implies \vec{T}_D\hat{j}+\vec{T}_C\hat{j}+\vec{T}_B\hat{j}=0$
  • $\Sigma{F}_z= 0 \implies \vec{T}_B\hat{k} - W=0$

    • Now you can solve for the tension in each cable.
