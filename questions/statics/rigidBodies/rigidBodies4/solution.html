First draw the reaction forces at point A.

The reaction forces at point A should be $A_x, A_y,A_z,M_y,M_z$.

Next we have to describe the tension in the cable as a vector.

We can do this by multiplying the the magnitude of the tension with the unit vector that matches the path of the cable.
    Finding the tension vector $\vec{T}$. First we must identify the unit vector that matches the path of the cable.
  • $\vec{U} = \frac{(L_2+L_2-L_1)\hat{i}-L_3\hat{j}+L_2\hat{k} }{\sqrt{(L_2+L_2-L_1)^2+(L_3)^2+(L_2)^2} }$
  • So the vector expression for the tesnion is simply $\vec{T} = (T)(\vec{U}) = (T)(\frac{(L_2+L_2-L_1)\hat{i}-L_3\hat{j}+L_2\hat{k} } {\sqrt{(L_2+L_2-L_1)^2+(L_3)^2+(L_2)^2} })$
    • Since we do not know what tesnion is yet we can just leave T in $\vec{T}$ for now. set up the equations that discribe all of the forces in the system. We will also need to take the moment about point A for an additional equation.
  • $\Sigma{F}_x = 0 \implies A_x + \vec{T}\hat{i} = 0$
  • $\Sigma{F}_y = 0 \implies A_y + \vec{T}\hat{j} = 0$
  • $\Sigma{F}_z = 0 \implies A_z + \vec{T}\hat{k} - W = 0$
    • Moment equations (about point A):

    But before we set up our three moment equations we first have to compute $\vec{d} X \vec{T}$.

    Where $\vec{d} = -2L_2\hat{i}+L_3\hat{j}+0\hat{k}$ goes from point A to the end of the cable.

  • $\vec{d} X \vec{T} = (L_3)(\vec{T}\hat{k})\hat{i}-(-2L_2)(\vec{T}\hat{k})\hat{j}+(-2L_2)(\vec{T}\hat{j})-(L_3)(\vec{T}\hat{i})\hat{k}$
    • setting up moment equations(about point A):
  • $\Sigma{M}_x = 0 \implies (L_3)(\vec{T}\hat{k})-(W)(L_3) = 0$
    • ;
  • $\Sigma{M}_y = 0 \implies -(-2L_2)(\vec{T}\hat{k})-(W)(L_2)+M_y= 0$
    • ;
  • $\Sigma{M}_z = 0 \implies M_z + (-2L_2)(\vec{T}\hat{j})-(L_3)(\vec{T}\hat{i})= 0$
    • ;

    Note: $\Sigma{M}_x$ does not have a moment reaction force since the structure can freely rotate about the x-axis.

  • So, $M_x = 0$
