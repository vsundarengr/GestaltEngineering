The first step is to compute the rection forces by using all of the external forces acting on the body. Also W = weight of person /2.

Solving for rection forces:

By using our static equilibrium force equations we can solve for $B_y$ and $A_y$. Since there is no friction we know $B_x$ and $A_x$ = 0.
  • By doing $\Sigma{M}_A = 0$ $\implies$ $W(L_2+L_1+\frac{L_1}{2})-B_y(L_1+L_1+\frac{L_1}{2}) =0$
  • Solving for $B_y$ we get $B_y = \frac{W(L_2+L_1+\frac{L_1}{2})}{L_1+L_1+\frac{L_1}{2} } $

  • By doing $\Sigma{F}_y = 0$ $\implies$ $ A_y + B_y -W =0 $
  • Solving for $A_y$ we get $ A_y = W-B_y$

      Using member CE to solve for $E_y$, $ E_x$, and $C_x$:

      FBD of CD:
    • On point E you can draw the reactions as up and left. On point C you can draw the reactions as down and left. Dont forget the $B_y$ reaction force on point B.

      • Solving:
    • By doing $\Sigma{F}_y = 0$ $ \implies$ $ B_y + E_y - C_y = 0 $
    • Solving for $E_y$ we get $E_y =C_y-B_y$

    • By doing $\Sigma{M}_C = 0$ $ \implies$ $ B_y(L_1)-E_y(xcos(\theta))-E_x(xsin(\theta)) = 0 $
    • Solving for $E_x$ we get $E_x = \frac{B_y(L_1)-E_y(xcos(\theta))}{xsin(\theta)}$
      • Where $\theta$ is the angle ECD, and x is ther length from C to E. You can find these values by doing some trig and geometry.

      Explanation:
    • Let angle A = angle CDE = $tan^{-1}(\frac{2L_1}{\frac{L_1}{2} })$
    • Let angle $\theta$ = angle ECD = $tan^{-1}(\frac{2L_1}{L_1})$
    • let angle C = angle DEC = 180 - (A+$\theta$)
    • Now let x = distance from C to E = $\frac{L_1sin(A)}{sin(C)}$

    • By doing $\Sigma{F}_x = 0$ $ \implies$ $-C_x-E_x = 0 $
    • Solving for $C_x$ we get $C_x = E_x$
